# Descrição

> O projeto tem como objetivo a criação do protótipo de um site voltado para a comunidade do jogo Red Dead Redemption 2.
> 
> O site irá contar com:
> * Aba de informações gerais
> * Aba de notícias acerca do jogo (Eventos, campeonatos, etc)
> * Aba de atualizações (Mudanças no jogo)
> * Aba de fórum (Sistema reddit de discussão com perguntas e respostas)
> * Aba de reports (Reportar bugs ou problemas)
>
> Público Alvo: Jogadores, Reviewers, Interessados
